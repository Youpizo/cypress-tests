# Installation de Cypress

1. **Installer Cypress** :
   Assurez-vous d'avoir Node.js et npm installés, puis exécutez la commande suivante pour installer Cypress dans votre projet :

   ```bash
   npm install cypress --save-dev


# Lancer cypress pour ouvrir l'interface graphique
npx cypress open


# Lancer les tests dans Chrome en mode HeadLess
npx cypress run --browser chrome

# Lancer les tests dans Firefox en mode HeadLess
npx cypress run --browser firefox
